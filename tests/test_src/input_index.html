Sphinx GitHub Pages Generator¶

Overview¶

This project provides a python library for accessing automatic generation of sphinx GitHub pages for a given repository. It is intended for th use with a Continuous Integration solutions. Developed using GitHub Actions.

In a Nutshell¶

Prerequisites¶

  • Python 3.8+

Installation¶

Documentation¶

#todo change link Documentation for the latest release is hosted on the Github Pages of this project.

Features¶

  • Build html documentation files using Sphinx

  • Choose which remote branch to generate the documentation from

  • Choose between automatically committing or committing and pushing the generated files to your GitHub Pages branch.

  • Works with multiple packages in one Repository.
